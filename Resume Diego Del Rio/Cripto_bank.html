Project CryptoBank

Pyhton Project

The project done on a cryptocurrency bank was done for the algorithm class using the Python programming language as well as numerous search and sort algorithms. During this project I improved my programming skills, my way of organizing myself when dealing with projects and my ability to think and solve problems. Thanks to this object oriented programming project I consider myself to have the necessary skills and ability in this programming language.

The project basically consisted of imitating every operation that a bank performs, currency deposit and withdrawal, transactions, exchange from one currency to another, opening an account, user management, currency creation... For this we used an initial csv with some users who already had a number of coins and after this and through object oriented programming we developed each of the roles (administrator and user) where each had a number of specific functions, the administrator can create coins, modify user information... something that the user can not do, we developed a hierarchy.

After having a clear idea of how the project was going to be approached, we moved on to the programming of the project and its corresponding testing. This link leads to my Github repository where I have uploaded the project.
