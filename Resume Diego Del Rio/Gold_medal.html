Data Mining Project

The project on predicting the number of gold medals a country will win at the Olympic Games was done for the data mining course using the Rstudio programming language, dedicated to statistical computing and graphics.

For this project we started from a database downloaded from the Internet, this database was unpurified and with values and data that did not serve us so we had to clean the database and add those values that we believed could benefit a country to obtain a greater or lesser number of gold medals. The database ended up having the following values: number of gold medals won by a country in 2008, whether or not it was the host of the 2008 Olympics, the GDP per capita of that year, the number of athletes sent to the games, the jetlag of each country in relation to the host, etc.

After performing statistical and normality analyses, we developed a model with the best possible fit, obtaining that the best model for winning golds was a function of the variables GDP per capita, country population, number of athletes, jetlag and whether the country is a host or not. We then performed correlations, decision plots, dendograms, residual error comparisons and confusion matrices. As a result, we obtained that our model had an accuracy of 91%.
